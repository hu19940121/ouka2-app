# v0.2.1 - 暗色主题

## 新增

- 新增暗色主题，覆盖主界面、顶部工具栏、侧边栏、电台表格、安装队列、底部播放器、日志面板和常用弹窗。
- 顶部工具栏新增主题切换按钮，使用月亮 / 太阳图标快速切换浅色与暗色主题。
- 主题偏好会保存到本机，下次打开应用时自动沿用上次选择。

## 改进

- 统一前端自定义样式与 Naive UI 主题变量，减少浅色 / 暗色模式之间的视觉割裂。
- 设置弹窗仍保留暗色主题开关，方便从偏好管理入口统一查看当前配置。
